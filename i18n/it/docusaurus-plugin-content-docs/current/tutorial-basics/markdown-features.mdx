# Italian Markdown Features

Docusaurus supporta **[Markdown](https://daringfireball.net/projects/markdown/syntax)** e alcune **funzionalità aggiuntive**.

## Argomento anteriore

I documenti Markdown hanno i metadati in alto chiamati [Front Matter](https://jekyllrb.com/docs/front-matter/):

```titolo del testo = "mio-doc.md"
// inizio evidenziazione
---
id: my-doc-id
titolo: Il mio titolo del documento
descrizione: la descrizione del mio documento
slug: /my-custom-url
---
// evidenzia-fine

## Intestazione Markdown

Testo di markdown con [link](./hello.md)
```

## Collegamenti

Sono supportati i collegamenti Markdown regolari, utilizzando percorsi di URL o percorsi di file relativi.

```md
Vediamo come [Creare una pagina](/create-a-page).
```

```md
Vediamo come [Creare una pagina](./create-a-page.md).
```

**Risultato:** Vediamo come [Creare una pagina](./create-a-page.md).

## Immagini

Sono supportate le normali immagini Markdown.

Aggiungi un'immagine in `static/img/docusaurus.png` e visualizzala in Markdown:

```md
![Logo Docusaurus](/img/docusaurus.png)
```

![Logo Docusaurus](/img/docusaurus.png)

## Blocchi di codice

I blocchi di codice Markdown sono supportati con l'evidenziazione della sintassi.

    ```jsx title="src/components/HelloDocusaurus.js"
    function HelloDocusaurus() {
        Restituzione (
            <h1>Ciao, Docusaurus!</h1>
        )
    }
    ```

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Ciao, Docusaurus!</h1>;
}
```

## Componenti MDX e React

[MDX](https://mdxjs.com/) può rendere la tua documentazione più **interattiva** e consente di utilizzare qualsiasi **componente React all'interno di Markdown**:

```jsx
export const Evidenzia = ({bambini, colore}) => (
  <span
    stile={{
      sfondoColore: colore,
      borderRadius: '20px',
      colore: '#ff',
      imbottitura: '10px',
      cursore: 'puntatore',
    }}
    onClick={() => {
      alert(`Hai cliccato sul colore ${color} con l'etichetta ${children}`)
    }}>
    {bambini}
  </span>
);

Questo è <Highlight color="#25c2a0">Docusaurus green</Highlight>!

```
